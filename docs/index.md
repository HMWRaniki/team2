🎮 オンラインマニュアル
＜ゲーム概要＞

本作は、シンプルで直感的な操作が魅力の
エアホッケーゲームです。

スピード感あふれる対戦の中で、
試合中に さまざまなアイテムがPOP し、
ゲーム展開が 大きく変化 します。

パックの 速度が上昇

ゴール範囲が変化

アイテムを活かした 戦略性のある駆け引き

ただ打ち返すだけではなく、
タイミングよくアイテムを使うことで逆転や一発逆転も可能。

💡 シンプルながらも毎試合違った展開を楽しめる、
新感覚のエアホッケーゲームです。

＜操作説明＞

本作は 同一キーボードで行う対戦形式 のゲームです。

■ 移動操作

それぞれ以下のキーでマレットを操作します。

← / A ：左移動

↑ / W ：上移動

→ / D ：右移動

↓ / S ：下移動

※ 操作は 移動のみ です。

■ ゲームルール

パックの操作・アイテム獲得は 自身のマレット で行う

相手側のゴールにパックを入れると1点

5点先取で勝利

⚠️ アイテムの使いどころが勝敗を左右します！
