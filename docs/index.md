オンラインマニュアル
＜ゲーム概要＞
本作は、シンプルで直感的な操作が魅力のエアホッケーゲームです。
スピード感あふれる対戦の中で、試合中にさまざまなアイテムがPOPし、ゲーム展開が大きく変化します。
パックの速度が上がったり、ゴール範囲が変化するなど、アイテムを活かした戦略が勝敗のカギを握ります。
ただ打ち返すだけではなく、タイミングよくアイテムを使うことで、逆転や一発逆転も狙える奥深いゲーム性が特徴です。
シンプルながらも毎試合違った展開を楽しめる、新感覚のエアホッケーを体験してください。

＜操作説明＞
同一キーボードで行う対戦形式のゲームで操作はそれぞれ、←Ａ、↑Ｗ、→Ｄ、↓Ｓでの移動操作のみ。
パックの操作やアイテム獲得は自身のマレットで行う。
相手側のゴールにパックを入れることができれば1点。
5点先取で勝利となる。
